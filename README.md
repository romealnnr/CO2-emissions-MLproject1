# CO2-emissions-MLproject1 

This project focuses on analyzing and predicting CO2 emissions from vehicles registered in Europe, using the 2024 dataset from the European Environment Agency (EEA). Transportation is a major contributor to greenhouse gas emissions, and predictive modeling can help stakeholders take proactive decisions.

Report 1 : 
The main objectives are:
•
building a robust and reproducible preprocessing and feature extraction pipeline
•
exploring ,analyzing and visualizing vehicle emission patterns
•
Identifying relevant technical features for CO₂ prediction
•
preparing the data for classical machine learning models
